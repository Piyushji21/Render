Endpoint Structure:

                GET /like?uid={USER_ID}&server_name={SERVER}&key=gst&like={OPTIONAL}
                
                
Parameters:

                
    
                    
  • uid (required) - User ID to send likes to
    • 
                    
  • server_name (required) - Server region (IND, BR, US, SAC, NA, BD)
    • 
                    
  • key (required) - API authentication key (currently: "gst")
    • 
                    
  • like (optional) - Number of tokens to use (omit for ALL tokens)
    • 
                

                
                
Example URLs:

                
/like?uid=123456789&server_name=IND&key=gst - Use ALL tokens

                
/like?uid=123456789&server_name=IND&key=gst&like=500 - Use 500 tokens

                
/like?uid=123456789&server_name=IND&key=gst&like=max - Use max tokens
